# 二维码生成器 - 快速开始指南

## 🚀 使用exe程序（推荐）

### 方法一：直接运行
1. 双击 `二维码生成器_GUI.exe`
2. 选择需要的二维码类型（网址/邮件/电话/短信/其他）
3. 填写相应信息
4. 点击"生成二维码"按钮
5. 查看预览，保存或另存为

### 方法二：命令行快速生成
```bash
二维码生成器_CLI.exe "https://github.com"
```

## 📱 各类型二维码使用示例

### 1. 网址二维码
**用途**：分享网页链接、应用下载页面

**操作步骤**：
1. 切换到"🌐 网址"选项卡
2. 输入完整URL（如：https://www.baidu.com）
3. 点击生成

**快速示例**：
- GitHub、百度、Google等预设按钮可一键填入

### 2. 邮件二维码
**用途**：名片、客服联系、意见反馈

**操作步骤**：
1. 切换到"📧 邮件"选项卡
2. 填写收件人邮箱（必填）
3. 填写抄送、主题、正文（可选）
4. 点击生成

**示例**：
- 收件人：support@company.com
- 主题：产品咨询
- 正文：您好，我想了解...

**效果**：扫描后自动打开邮件应用，收件人和内容已填好

### 3. 电话二维码
**用途**：名片、客服热线、紧急联系

**操作步骤**：
1. 切换到"📞 电话"选项卡
2. 输入电话号码（如：+8613800138000 或 10086）
3. 点击生成

**效果**：扫描后自动打开拨号界面，点击即可拨打

### 4. 短信二维码
**用途**：活动报名、验证码获取、快速反馈

**操作步骤**：
1. 切换到"💬 短信"选项卡
2. 输入收件人号码（必填）
3. 输入短信内容（可选）
4. 点击生成

**示例**：
- 收件人：10086
- 内容：查询余额

**效果**：扫描后打开短信应用，号码和内容已填好

### 5. WiFi二维码
**用途**：分享WiFi密码、公共场所WiFi

**操作步骤**：
1. 切换到"⚡ 其他"选项卡
2. 找到"📶 WiFi连接"区域
3. 输入网络名称（SSID）
4. 输入密码
5. 选择加密类型（WPA/WPA2、WEP、无）
6. 点击"生成WiFi二维码"

**效果**：扫描后自动连接WiFi（部分手机支持）

### 6. 地理位置二维码
**用途**：地址分享、活动地点、导航

**操作步骤**：
1. 切换到"⚡ 其他"选项卡
2. 找到"📍 地理位置"区域
3. 输入纬度和经度
4. 点击"生成位置二维码"

**示例**：
- 纬度：39.9042（天安门）
- 经度：116.4074

**效果**：扫描后打开地图应用并定位

## ⚙️ 二维码设置说明

### 大小
- **5**：小尺寸，适合打印在小物件上
- **10**：标准尺寸（推荐）
- **15**：大尺寸，适合远距离扫描
- **20**：超大尺寸，适合海报

### 容错率
- **L (7%)**：低容错，二维码更简洁
- **M (15%)**：中等容错
- **Q (25%)**：较高容错
- **H (30%)**：高容错（推荐），即使部分损坏也能扫描

## 💡 实用技巧

### 1. 批量生成（CLI版本）
创建 `urls.txt` 文件，每行一个URL：
```
https://www.baidu.com
https://www.google.com
https://github.com
```

运行CLI程序，选择选项2，输入文件路径即可批量生成。

### 2. 窗口调整
- GUI窗口支持自由调整大小
- 内容会自动适应窗口宽度
- 可以滚动查看所有选项

### 3. 文件命名
- 默认使用时间戳命名（如：qrcode_20250126_020000.png）
- 可以自定义文件名
- 所有二维码保存在 `qr_codes/` 目录

### 4. 保存和分享
- 点击"另存为"可保存到任意位置
- 点击"打开文件夹"可快速查看生成的二维码
- 支持直接拖拽到其他应用

## ❓ 常见问题

**Q: 扫描二维码后没有反应？**
A: 请确保输入的内容格式正确，网址需要以 http:// 或 https:// 开头。

**Q: 二维码扫不出来？**
A: 尝试增加"大小"参数或提高"容错率"。

**Q: 如何生成应用深链接？**
A: 在网址选项卡直接输入，如 weixin://（打开微信）、alipay://（打开支付宝）。

**Q: WiFi二维码不能自动连接？**
A: 部分手机不支持WiFi二维码自动连接，需要手动输入密码。

**Q: 可以生成彩色二维码吗？**
A: 当前版本仅支持黑白二维码，保证最佳兼容性。
